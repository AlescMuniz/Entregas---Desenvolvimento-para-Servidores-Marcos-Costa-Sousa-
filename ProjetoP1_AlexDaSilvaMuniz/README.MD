# ProjetoP1 - Sistema PetShop 🐾

Projeto desenvolvido para a P1 da disciplina Desenvolvimento para Servidores I
Fatec São Roque - Professor Marcos Costa de Sousa

**📋 Sobre o Projeto**
Sistema simples de PetShop desenvolvido em Java, aplicando os conceitos de Orientação a Objetos vistos em aula.


**▶️ Como Executar**

Abrir o projeto no NetBeans (ou VSCODE)
Executar a classe TestePetShop.java com Shift + F6
